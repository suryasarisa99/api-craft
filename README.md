# Api Craft
- Alternative to Postman, Insomnia, Bruno, Yaak
- it is inspired from Yaak.


**supports**
- http/https, websocket, graphql
- support template functions

![preview](./images/preview.png)



## Template Functions

- `response.body.path`
- `response.body.raw`
- `response.header`
- `cookie.value`
- `regex.match`
- `regex.replace`
- `json.path`
- `xml.path`
- `prompt.text`
